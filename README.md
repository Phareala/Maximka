# Мессенджер Максимка

Проект разложен по папкам:

- `frontend/` — HTML, CSS, JS
- `backend/` — Flask API, SQLite

## Функционал

- регистрация и вход
- шапка только после авторизации
- личные чаты
- создание групп
- поиск пользователей по логину и отображаемому имени
- непрочитанные сообщения и бейджи
- отправка текста и файлов
- предпросмотр фото и видео перед отправкой
- профиль с просмотром текущей и новой аватарки
- системный статус: онлайн / не беспокоить / оффлайн
- настройки темы, шрифта, плотности и уведомлений
- страницы "Справка" и "О приложении"

## Как запустить

```bash
cd backend
python -m venv .venv
```

Windows:

```bash
.venv\Scripts\activate
```

Linux/macOS:

```bash
source .venv/bin/activate
```

```bash
pip install -r requirements.txt
python app.py
```

Сервер стартует на `http://127.0.0.1:5000`.
